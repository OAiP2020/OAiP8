 5. Main Topic 
Действительно случайные числа
Решением  для  генерации  действительно  случайных  чисел  является   использование текущего  времени  в  качестве  значения  переменной   seed  для  функции  srand().
В следующем примере используется функция time() для получения количества секунд в системных часах вашего компьютера, и  включение функции rand() со случайным значением  seed. При этом, должна быть подключена библиотека  ctime  (заголовочный файл <ctime>).  
time()  вернет  текущее  количество  секунд,  побуждая  функцию     srand()  передавать  разное  значение   seed  для  функции   rand()  при  каждом  запуске  программы.

 
В  данном  случае  при  каждом  запуске  программы  будут  выводится  разные  значения.
