 5. Main Topic 
Определение функции-1
Для  определения  функции  в  языке  C++  используется следующий  синтаксис:
Здесь 
return_type (возвращаемый  тип) - тип  данных,  возвращаемый  функцией.
function_name - имя  функции.
parameter list (список параметров) - при  вызове  функции  вы передаете  значение  в  параметр. Это  значение  ссылается  на актуальнпый параметр  или  аргумент.  Для  каждого   параметра указываются  его  тип  и  имя.
body of the function (тело функции) - набор  операторов,  определяющий  что  делает  функция.

 
В С++ допустимо написание функций без  параметров.
